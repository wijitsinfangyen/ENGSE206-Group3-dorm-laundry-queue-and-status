# Evidence Repository

ใช้เก็บหลักฐานที่สนับสนุนการตัดสินใจของกลุ่ม เช่น

- ภาพ/บันทึกจาก workshop หรือ role-play
- interview notes ที่ **anonymized** แล้ว
- screenshot ของ prototype test / peer review
- feedback form หรือ summary ที่ไม่เปิดเผยข้อมูลส่วนบุคคล
- exported PDF/PNG ของ diagram ที่อ้างอิงในเอกสาร

## กติกาความเป็นส่วนตัว

- ห้ามใส่ข้อมูลส่วนบุคคลจริง เช่น ชื่อ-นามสกุล เบอร์โทร อีเมล รหัสนักศึกษา หรือภาพใบหน้าโดยไม่จำเป็น
- ตั้งชื่อไฟล์แบบ `week-04-interview-summary-01.md` หรือ `week-15-usability-note-02.md`
- หลักฐานต้องถูกอ้างอิงกลับไปยัง `E-ID` ใน `docs/04-evidence-log.md` เมื่อเกี่ยวข้อง
