# Project Management Workspace

เอกสารในโฟลเดอร์นี้ช่วยให้เห็น **การมีส่วนร่วมของสมาชิก การตัดสินใจ ความเสี่ยง และความก้าวหน้า** โดยไม่ซ้ำกับเอกสาร requirements/design

- `team-worklog.md` — ทุกคนบันทึกงานและเวลา/หลักฐานอย่างกระชับ
- `meeting-minutes/` — บันทึกการประชุมและมติ
- `decision-log.md` — การตัดสินใจสำคัญที่มี rationale
- `risk-and-issue-log.md` — ความเสี่ยงและปัญหาที่ต้องติดตาม
- `backlog.md` — งานถัดไปของทีม
- `ai-use-log.md` — บันทึกการใช้ AI ที่มีผลต่อชิ้นงาน
